실습 도구 비용
01 · Workflow

Gems → Opal → 나만의 웹앱

Gems로 AI에게 역할·말투·지식 범위를 설정하고, Opal 노드 에디터로 입력·생성·출력을 연결하면 프롬프트 한 줄로는 만들 수 없던 상호작용하는 앱이 완성됩니다.

STEP 1
💎

Gems 페이지

자연어로 "어떤 비서가 필요한지" 서술합니다. 예: "4학년 과학 수업에서 식물의 한살이를 설명하는 친절한 선생님"

🟦 input: 학생 질문
🤖 persona: 다정한 과학 선생님
📚 knowledge: 4-2학기 과학 교과서
STEP 2
🔮

Opal 노드 에디터

입력 → 생성 → 출력을 시각적으로 연결합니다. 3가지 색으로 구분된 노드를 끌어다 놓으면 완성.

🟡 Input → 🔵 Generate → 🟢 Output
STEP 3
✨

앱 미리보기

만든 앱이 실제로 동작하는 미리보기를 확인. 학생에게 URL 또는 QR로 공유하면 끝.

✅ 즉시 완성
배포·로그인·결제 없이 수업 바로 투입
02 · Tools

오늘 실습에서 쓸 두 가지 도구

설치·로그인·API 키 없이 브라우저만으로 바로 체험할 수 있습니다.

🔮
TOOL 01 · 인터랙티브

SuperGems Simulator

Gems + Opal Editor를 체험할 수 있는 웹 시뮬레이터. 설치 없이 브라우저에서 노드 에디터를 직접 움직여보며 "자연어 설명 → 시각적 워크플로 → 실제 앱"의 변환을 배웁니다.

  • ▸ Gems 페이지: 원하는 AI 미니앱을 자연어로 서술
  • ▸ Opal 에디터: 🟡 Input → 🔵 Generate → 🟢 Output 노드 연결
  • ▸ 앱 미리보기: 완성된 앱의 실시간 시뮬레이션
  • ▸ 단일 HTML 파일 · 오프라인 동작 · 100% 한국어

03 · Scenarios

2회차 실습 시나리오

포스터에 공지된 두 개 실습 주제. 오늘 2시간 30분 안에 두 가지 모두 완성합니다.

🤝
실습 ①

AI 전담 비서 Gems 맞춤 설정

내 반 · 내 과목 · 내 교수 스타일에 맞춘 AI 비서를 만듭니다. 페르소나 · 말투 · 지식 범위 · 금지 사항까지 세세하게 튜닝.

• 역할 정의 (선생님 · 튜터 · 상담자)
• 학년 눈높이 말투 (초등 저/고 · 중·고)
• 교과 자료 · 금지 주제 주입
• 학생 테스트 · 반복 수정
🎨
실습 ②

비주얼 학습자료 생성

Opal + Gems로 학생이 입력하면 그림·도표·인포그래픽으로 설명하는 미니앱을 제작. 글로만 설명하던 개념을 학생 눈높이 비주얼로 즉시 변환.

• 입력: 학생 질문 · 개념
• 생성: Imagen · 도표 프롬프트
• 출력: 카드형 비주얼 + 설명
• 학생에게 URL/QR 공유

실습 준비물 체크

제미나이 유료 계정 필수 · 개인 G메일로만 (학교 계정 불가) · Chrome/Edge 최신 버전 권장 · 실습 전 시뮬레이터를 먼저 둘러보시면 이해가 빨라집니다.
