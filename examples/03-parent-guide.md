# 예제 03 — 학부모 안내 앱

> **시나리오**: 현장학습 · 학사일정 · 건강 설문을 한곳에서 보는 학부모 전용 앱
> **소요 시간**: 50–70분
> **난이도**: ⭐⭐⭐ 고급 (폼 처리)

## 프롬프트

````text
@SKILL.md
@presets/education/parent-formal.md 참조.
DIALS: VARIANCE=3, MOTION=3, DENSITY=5

# 프로젝트: 한강초 4-3 학부모 안내 앱

## 목적
학부모가 학사 일정·건강 설문·담임 공지를 한 페이지에서 확인·응답.
공문 감각 유지하되, 휴대폰에서 편하게 읽히는 현대적 가독성.

## 사용자 플로우
1. [홈] 최신 공지 3건 + 다가오는 일정 3건 + 체크해야 할 설문 1건
2. [일정] 월력 뷰 + 리스트 뷰 전환
3. [설문] 건강상태·참여의사 등 설문 목록 + 응답 폼
4. [문의] 담임 연락 · FAQ

## 홈 화면 스펙
- 상단 헤더: "한강초등학교 4학년 3반 · 담임 김○○"
- 인사 블록: "안녕하십니까. 오늘 날씨는 맑음입니다."
- 최신 공지 카드 3개:
  - 카드: 배지(중요/일반) · 제목 · 발행일 · 본문 미리보기 3줄 · "자세히"
- 다가오는 일정 타임라인:
  - 세로 타임라인 · 날짜 배지 + 제목 + 장소
  - 예: "5.28 (화) | 봄 현장학습 | 국립중앙박물관"
- 응답 대기 설문 배너:
  - 옅은 노랑 배경 + "현장학습 참가 동의서를 작성해 주세요" + "응답하기" CTA

## 설문 폼 스펙
- 라디오·체크박스·단답형·장문형 조합
- 각 문항 위에 설명 텍스트 · 필수 표기(*)
- 제출 버튼 하단 고정(sticky)
- 제출 후 성공 화면: "제출되었습니다. 확인 메일을 확인해 주세요."

## 일정 화면 스펙
- 상단 토글: [월력] [리스트]
- 월력 뷰: 간단한 달력 그리드, 일정 있는 날은 색 점
- 리스트 뷰: 날짜별 그룹, 아이콘 + 제목 + 장소 + 더보기 모달

## 디자인 규칙
- 색: 딥 네이비 `#1E2A4B` 중심. 강조는 진짜 중요할 때만 빨강
- 폰트: Pretendard Variable 전면 (공식 감각)
- 카드: 라운드 8px · 테두리 1px · 그림자 최소
- 아이콘: Iconify Solar (solar:calendar-bold-duotone 등)
- 이모지 사용 금지 (공식 문서)
- 애니메이션: 페이드인만, 0.3s 이내

## 콘텐츠 규칙
- 학생 실명 없음 (학번·별명만)
- 발신 주체 명시: "4학년 3반 담임 김○○ 드림"
- 격식체: "~하시기 바랍니다", "~드립니다"
- 긴급 공지는 배지로만 표시

## 기술
- 단일 `index.html` (단순 버전) 또는 Vite+React (확장 버전)
- 폼 제출은 Google Form 연동 또는 Tally.so 임베드
- LocalStorage로 읽은 공지 표시 제거

## 접근성
- 모든 링크 `aria-label`
- 프린트 CSS 대응 (학부모가 출력할 수 있음)
- 고대비 모드 호환
- 60대 학부모도 편한 폰트 크기 (본문 17+)

## 산출물
- `index.html` 혹은 Vite 프로젝트
- 예시 데이터 3–5건
- Google Form 연결 자리 주석
````

## 주의 사항

- 학부모 개인정보 수집은 **학교 공식 채널**로 한정 — 이 앱에서는 직접 저장하지 않고 Google Form 연결만
- 설문 결과 열람은 교사만 (별도 비공개 페이지)
