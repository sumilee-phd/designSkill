# 예제 05 — 출결 확인 대시보드

> **시나리오**: 담임이 매일 아침 빠르게 출결을 체크하는 교사 전용 도구
> **소요 시간**: 40–60분
> **난이도**: ⭐⭐ 중급

## 프롬프트

````text
@SKILL.md
@presets/korean/toss.md 참조 (숫자·대시보드 감성)
DIALS: VARIANCE=3, MOTION=2, DENSITY=7

# 프로젝트: 4학년 3반 출결 대시보드

## 사용자
담임 교사 전용. 월~금 아침 9시에 1분 안에 체크 완료할 수 있어야 함.

## 주요 화면

### 1. 오늘의 출결
- 상단 요약: "2026.04.23(수) · 출석 24 · 결석 1 · 지각 1 · 조퇴 0"
- 학생 리스트 (아바타 + 이름 + 상태 뱃지)
- 탭 한 번으로 상태 전환: 출석 → 지각 → 결석 → 조퇴
- 저장 버튼 (하단 sticky)

### 2. 주간 요약
- 요일별 막대그래프 (출석률 %)
- 결석이 많은 학생 TOP 3 (개별 연락 필요)
- "지난주 대비 +2% 개선" 같은 인사이트

### 3. 월간 기록
- 달력 뷰 · 각 날짜 셀에 출석률 색 그라데이션
- 클릭 시 해당 날짜 상세

### 4. 학생별 기록
- 학생 선택 → 그 학생의 출석 이력 타임라인
- 연속 출석·결석 기록 · 지각 빈도

## UI 스펙

### 출결 리스트 아이템
```
[아바타]  이름 (학번)        [출석 ✓] [지각] [결석] [조퇴]
          마지막 기록: 어제 출석
```
- 기본 상태: 출석 (탭하면 회전)
- 탭 한 번으로 다음 상태로 → 길게 누르면 메뉴 (사유 입력)

### 통계 카드
- 큰 숫자(Pretendard Black 64px) + 라벨 + 전주 대비 변화
- 색: 출석 그린 / 결석 레드 / 지각 옐로우

### 사유 입력 (결석·지각 시)
- 모달: 라디오 (병결 · 경조사 · 가정사유 · 기타) + 메모 (선택)

## 기술
- Vite + React + Tailwind + shadcn/ui
- LocalStorage로 상태 저장 (백엔드 없이 실습 가능)
- 엑셀 내보내기 기능 (`xlsx` 라이브러리 · 월말 출결 정리용)
- PWA 기본 (홈화면 추가 가능)

## 교사 친화 기능
- 키보드 단축키: 숫자키 1=출석 · 2=지각 · 3=결석 · 4=조퇴
- 스와이프 제스처 (모바일)
- 실수 방지: 저장 전 확인 토스트
- 어제 상태 자동 로드 (빨리 체크 후 수정만)

## 접근성
- 대비 높게 (교실 조명·햇빛 방해 고려)
- 터치 타겟 56×56
- 색만으로 상태 구분 금지 → 아이콘·텍스트 병행

## 산출물
- Vite React 프로젝트
- 학생 더미 데이터 25명
- 엑셀 내보내기 기능 동작
````
