# 예제 04 — 학생 포트폴리오

> **시나리오**: 중등 학생이 1년간의 프로젝트·작품·활동을 정리한 개인 웹 포트폴리오
> **소요 시간**: 60–90분
> **난이도**: ⭐⭐⭐ 고급

## 프롬프트

````text
@SKILL.md
@presets/global/notion.md 참조 (에디토리얼 느낌 원할 때)
또는 @presets/education/student-vibrant.md (개성 강조 원할 때)
DIALS: VARIANCE=7, MOTION=5, DENSITY=4

# 프로젝트: 중학생 연간 포트폴리오

## 목적
- 중학교 3학년 학생이 1년간 수행한 프로젝트·작품·수상 경력을 정리
- 고교 진학·동아리 지원 시 URL로 공유 가능한 개인 웹 포트폴리오
- 익명성 선택 가능(실명 vs 닉네임)

## 페이지 구조
1. Hero
   - 이름 또는 닉네임 + 한 줄 소개 + 핵심 키워드 태그 3–5
   - 프로필 사진(선택) 또는 이니셜 아바타
   - CTA: "프로젝트 보기" · "연락하기"

2. About
   - 관심 분야 · 취미 · 미래 희망
   - 100–300자 자기소개

3. 프로젝트 (주요 섹션)
   - 카드 그리드 (2–3열)
   - 각 카드: 대표 이미지 · 제목 · 기간 · 역할 · 한 줄 요약 · 태그
   - 카드 클릭 → 상세 모달 또는 상세 페이지
   - 상세: 배경 · 과정 · 결과 · 회고 · 관련 이미지 갤러리

4. 작품 갤러리 (선택)
   - 그림 · 사진 · 음악 · 영상 임베드

5. 수상·활동 이력
   - 타임라인 형식
   - 2025.03 학교 과학 경진대회 장려상
   - 2025.06 OO 독서 동아리 부장
   - ...

6. 스킬 · 도구
   - 아이콘 + 스킬명 + 숙련도 바 (선택)

7. 연락
   - 이메일 · GitHub · 인스타 등 선택 공개
   - 쪽지 폼 (선택)

## 디자인 옵션

### 옵션 A: Notion 스타일 (에디토리얼)
- 모노크롬 · 본문 폭 680px · 리스트·블록 중심
- 사이드바 목차 고정 (데스크톱)
- 글자 중심, 이미지는 보조

### 옵션 B: 학생 바이브런트 (개성 강조)
- 비비드 컬러 · 거대 타이포 · 비대칭
- 프로젝트 카드를 비정형 배치
- 스크롤 기반 애니메이션

## 콘텐츠 샘플
- 프로젝트 예시:
  ```
  제목: "우리 학교 급식 만족도 조사 인포그래픽"
  기간: 2025.05 ~ 2025.07
  역할: 데이터 수집 · 시각화 · 발표
  요약: 300명 대상 설문조사 결과를 D3.js로 시각화. 교장선생님께 제안서 전달.
  태그: #데이터시각화 #학교프로젝트 #D3js
  ```

## 기술
- Vite + React + Tailwind (확장성 고려)
- 또는 단일 HTML (간단 버전)
- 다크/라이트 토글 제공 (고학년이니까)
- 프로젝트 데이터는 `projects.json` 으로 분리

## 접근성
- 모든 이미지 alt (그림이라면 그림 설명)
- 색 대비 AA 이상
- 동영상 임베드는 자동재생 OFF

## 산출물
- Vite 프로젝트 또는 단일 HTML
- `projects.json` 예시 5건
- 배포 후 URL 공유 예: `https://hong-gildong-portfolio.vercel.app`
````
