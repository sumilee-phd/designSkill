# 프로젝트: 한강초 2026 학예회 초대장

## 목적
학생·학부모·지역 주민을 학예회에 초대. 일정·프로그램·참석 응답 받기.
감각 있고 기억에 남는 초대장.

## 페이지 구조 (단일 HTML)

### 1. Hero (풀스크린)
- 대형 타이틀: "한강 2026" + "학예회에 초대합니다"
- 날짜·시간·장소 (큰 글씨)
  - 2026.11.15 (금) 18:00
  - 한강초등학교 대강당
- CTA: "참석 응답하기" · "프로그램 자세히"
- 배경: 조명·무대 느낌 그라디언트 또는 Lottie 애니메이션

### 2. 프로그램 타임라인
- 순서별 카드 (학년별 공연)
- 카드: 팀명 · 공연명 · 시간 · 장르 태그
- 예:
  - 17:50 개회사 · 교장선생님 인사말
  - 18:00 1학년 합창 "봄이 오면"
  - 18:15 2학년 무용 "한여름 밤의 꿈"
  - 18:30 3학년 연극 "마을의 이야기"
  - ...
  - 20:30 폐회 · 기념촬영

### 3. 출연진 갤러리 (익명)
- 학년·팀별 실루엣 아바타 + 팀명
- 호버 시 공연 소개 팝업

### 4. 오시는 길
- 지도(Kakao 지도 iframe 또는 정적 이미지)
- 주차 안내 · 대중교통

### 5. 참석 응답 폼
- 이름(학부모/지역주민/학생) · 연락처 · 동반 인원
- 제출 → 확인 화면 + 카운트다운 위젯 표시 (D-7)

### 6. 공유
- 카카오 공유 · 링크 복사 · QR 생성
- SNS 해시태그 안내

## 디자인 규칙
- 학생 바이브런트 스타일: 핫핑크·일렉트릭 블루·형광 그린
- 거대 타이포 (Display 96px)
- 컬러 블록 카드 · 오프셋 섀도우
- 스크롤 기반 패럴랙스 (Hero → 프로그램 전환 시)
- 모션: 타이머 카운트다운 (D-?)

## 기술
- 단일 `index.html` + Tailwind CDN + Pretendard CDN
- 폼은 Google Form 임베드
- 카운트다운: JS 1줄
- 카카오 공유 SDK (선택)

## 접근성
- 화려한 애니메이션이지만 `prefers-reduced-motion` 존중
- 색 대비는 WCAG AA
- 대강당 위치·접근성 정보 제공
