# 예제 07 — 독서록 앱

> **시나리오**: 학생이 한 해 동안 읽은 책을 기록·회고하는 개인 독서 앱
> **소요 시간**: 50–70분
> **난이도**: ⭐⭐ 중급

## 프롬프트

````text
@SKILL.md
@presets/global/notion.md 참조 (에디토리얼)
DIALS: VARIANCE=4, MOTION=3, DENSITY=4

# 프로젝트: 학생 독서록 앱

## 목적
학생이 읽은 책을 기록하고, 한 문장 감상·별점·인용구를 남긴다.
학기 말 통계(총 권수 · 장르 분포 · 인상 깊은 구절)로 회고.

## 사용자 플로우
[홈: 최근 읽은 책] → [책 추가] → [상세 편집] → [통계]

## 화면 스펙

### 홈
- 상단: "올해 읽은 책 N권 · 평균 ★ ?.?" 요약
- 최근 책 카드 가로 스크롤 (3–5권)
- "독서 스트릭" 달력형 히트맵 (Github 기여도 느낌 · 초록 톤)
- CTA: "+ 책 추가" FAB

### 책 추가 모달
- 책 제목 · 저자 · 장르(드롭다운) · 시작일 · 종료일
- 별점(★×5 스와이프)
- 한 줄 감상 (140자 제한)
- 인상 깊은 구절 (장문형, 선택)
- 표지 이미지 업로드 또는 URL

### 상세 페이지 (책 하나당)
- 표지(큰 이미지) · 제목 · 저자 · 별점
- 읽은 기간 · 장르 태그
- 한 줄 감상 (인용 스타일)
- 인상 깊은 구절 (블록 인용)
- "비슷한 책 추천" (선택 기능)

### 통계 페이지
- 월별 읽은 권수 막대그래프
- 장르 분포 도넛 차트
- 평균 별점
- "가장 오래 기억에 남는 한 줄" (별점 5점 중 랜덤)

## 디자인
- 본문 폭 680px · 모노크롬 + 장르별 은은한 색 태그
- 서체 Pretendard + 인용은 세리프(Noto Serif KR) 혼합
- 책 표지는 라운드 6px · 살짝 그림자
- 밝은 모드 + 다크 모드 토글

## 데이터 구조

## 기술
- Vite + React + Tailwind + shadcn/ui
- LocalStorage (또는 Supabase 연동 확장)
- 차트: Recharts 또는 바닐라 SVG
- 표지 이미지: Kakao Book API · Google Books API (선택)

## 산출물
- Vite 프로젝트
- 예시 책 8권 더미 데이터
- 배포 후 개인 독서 아카이브로 사용 가능
````
