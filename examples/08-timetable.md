````text
@SKILL.md

DIALS: VARIANCE=4, MOTION=3, DENSITY=8

# 프로젝트: 교사용 주간 시간표

## 목적
담임·교과 담당 교사가 한눈에 주간 수업을 확인.
강의실 변경·임시 조정·대체 수업 메모 가능.

## 핵심 화면

### 1. 주간 그리드 뷰 (기본)
- X축: 월~금 · Y축: 1–7교시
- 셀에 수업명 · 반 · 교실
- 현재 시간 하이라이트 (빨간 가로선)
- 셀 클릭 → 상세 모달
- 색 코딩: 과목별 고유 색 (국영수 기본, 교사가 커스텀)

### 2. 오늘 뷰
- 상단 날짜 + 요일
- 세로 타임라인 (9:00–15:00)
- 각 교시 카드: 수업명 · 반 · 교실 · 준비할 것 메모
- 현재 진행중 강조 (왼쪽 파란 막대)

### 3. 교실 가용성
- 교실별 빈 시간 표시 (보강 수업·학부모 상담 잡을 때)

### 4. 빠른 메모
- "4반 영어 교실 3 → 5로 임시 변경" 같은 메모 · 날짜 범위 · 자동 반영

## UI 스펙

### 그리드 셀
```
┌─────────────────┐
│ 4-3 국어   [3관] │  ← 수업명 + 반 + 교실
│ 봄 시단원 2차시  │  ← 진도 메모
└─────────────────┘
```
- 빈 교시: 옅은 배경, "자율" 또는 "공강"
- 연속 교시: 블록으로 병합 표시

### 상세 모달
- 수업 정보 · 학생 수 · 지난 시간 진도 · 준비물
- "임시 변경" 버튼 → 해당 회차만 수정
- "보강 공지 보내기" (학생에게 단톡방 메시지 — Kakao Work 연동 선택)

## 디자인
- 다크 모드 기본 (컴퓨터실·야간 사용 빈번)
- 과목별 색: 국어 빨강 / 영어 파랑 / 수학 보라 / 과학 초록 / 사회 주황
- 색만으로 구별 안 함 — 텍스트·아이콘 병행
- 정보 밀도 높음 (DENSITY=8) — 여백 최소

## 기술
- Vite + React + Tailwind
- 시간표 데이터 `timetable.json`
- 임시 변경 기록 LocalStorage
- 출력: 프린트 CSS로 A4 1장

## 교사 친화 기능
- 학기 시작 시 엑셀 업로드로 시간표 한 번에 가져오기
- 자동 충돌 감지 (같은 시간 두 수업 배정 시 경고)
- 단축키: ←→ 로 주 단위 이동, T로 오늘

## 산출물
- Vite 프로젝트
- 예시 시간표 (주간 전체)
- 엑셀 import 샘플 `.xlsx`
````
