# 예제 10 — 교과 자료 허브

> **시나리오**: 한 학기 동안의 과목별 수업 자료를 모아 학생에게 공유하는 허브 사이트
> **소요 시간**: 60–90분
> **난이도**: ⭐⭐⭐ 고급 (멀티페이지)

## 프롬프트

````text
@SKILL.md
@skills/11-stitch-multipage 호출
@presets/education/lesson-clean.md 참조
DIALS: VARIANCE=4, MOTION=3, DENSITY=5

# 프로젝트: 4학년 3반 교과 자료 허브

## 목적
국어·수학·사회·과학·영어 등 여러 과목 수업자료·영상·숙제를 한곳에서
학생이 찾아볼 수 있는 허브. 과목별 페이지 + 검색·태그 지원.

## 사이트맵
```
index.html         — 메인 (과목 6개 타일 + 최근 업데이트)
korean.html        — 국어
math.html          — 수학
social.html        — 사회
science.html       — 과학
english.html       — 영어
about.html         — 소개 (담임 + 수업 철학)
404.html
```

## 메인 페이지 스펙

### Hero
- "4학년 3반 수업자료 허브"
- 서브: "언제든 다시 보고 싶은 수업, 여기에 모아뒀어요"
- 검색창 (단원·키워드)

### 과목 타일 (6개 그리드)
- 각 과목 색 코딩
- 아이콘 + 과목명 + "자료 N개" · 최근 업데이트 날짜
- 클릭 → 해당 과목 페이지

### 최근 업데이트 리스트 (10개)
- 과목 배지 · 제목 · 단원 · 날짜
- 예: "[과학] 식물의 한살이 3차시 · 발표자료 · 5.20"

### 태그 클라우드
- 인기 태그: "발표자료", "영상", "숙제", "실험", "독서"

## 과목별 페이지 스펙 (예: 과학)

### Hero
- 과목명 · 담당 선생님 · 이번 학기 단원 수
- "단원별 자료" 바로가기 앵커

### 단원별 섹션 (아코디언)
- 단원 1: 식물의 한살이
  - 1차시: 씨앗 관찰 (PDF) · (영상) · (활동지)
  - 2차시: 싹 틔우기 실험
  - 3차시: 발표자료
- 단원 2: 힘과 운동
  - ...

### 자료 카드
- 아이콘(PDF/영상/PPT/이미지) · 제목 · 업로드일 · "다운로드" 버튼

### 숙제·과제 섹션
- 마감일 배지 (D-3 등)
- 체크리스트형

## 데이터 구조
```js
const resources = [
  {
    id: 'r-001',
    subject: 'science',
    unit: '식물의 한살이',
    session: 1,
    type: 'pdf',
    title: '씨앗 관찰 활동지',
    uploadedAt: '2026-04-15',
    url: '/files/plant-seed.pdf',
    tags: ['관찰', '활동지']
  }
  // ...
];
```

## 디자인
- 수업자료 클린 스타일 (딥 블루 · 본문 폭 넓게 · 정보 밀도)
- 과목 색: 국어 빨강 / 영어 파랑 / 수학 보라 / 과학 초록 / 사회 주황
- 아이콘: Iconify Solar
- 정적 검색: JS에서 필터링 (백엔드 없이)

## 기술
- 단일 HTML 다중 페이지 (Vite 없이도 가능)
- 공통 header/footer 조각 include (`<script>` 로 삽입 또는 hand-copy)
- `resources.json` 분리
- 검색: MiniSearch 또는 Fuse.js (가벼움)
- 프린트 CSS 대응

## 접근성
- 모든 파일 다운로드에 확장자·크기 표시
- 영상 임베드는 자막 확인 권고
- 키보드 탐색 완전 지원

## 산출물
- 7페이지 HTML 세트 + `resources.json` + 공통 스타일
- 예시 자료 30개 이상
- 배포 후: `https://class-4-3-hub.vercel.app`
````
