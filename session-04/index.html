15 · ★★ 4회차 핵심 Google AI Studio Build

ai-studio-build — UI에 실제 기능 이식

Stitch가 만든 UI는 그림일 뿐입니다. 버튼 클릭·폼 제출·Gemini API 연동 같은 실제 동작은 AI Studio Build에서 구현합니다. Stitch의 디자인을 그대로 유지하면서 React+TypeScript+Tailwind 앱으로 완성 → GitHub 자동 연동 → Vercel 배포까지 한 흐름으로.

① UI 이식
Stitch HTML + DESIGN.md 첨부
② 기능 구현
폼 · Gemini API · LocalStorage
③ GitHub 자동 연동
코드 자동 커밋 · Vercel 연결

연수 4회차 워크플로

2026.05.21 (목) 19:00–21:30 · 아이디어에서 실제 배포까지 한 흐름으로 진행합니다.

  1. 1

    AI Studio에서 아이디어 발상 · 기능 명세

    구현할 앱의 목적·사용자·핵심 기능을 Gemini와 정리합니다.

  2. 2

    enhance-prompt · korean-style · 프리셋 선택

    취지에 맞는 프리셋을 골라 프롬프트를 강화합니다.

  3. 3

    Stitch에서 화면 생성 · DESIGN.md 추출

    첫 번째 화면을 만들고, design-md로 일관성 문서를 확보합니다.

  4. 4

    stitch-multipage로 여러 화면 완성

    다중 페이지 사이트를 확장해 완결된 UI 흐름을 만듭니다.

  5. 5

    AI Studio Build로 기능 구현 ★ 핵심

    Stitch가 만든 UI에 실제 동작(폼 제출·데이터 저장·Gemini API 연동)을 넣어 "진짜 앱"으로 완성합니다.

  6. 6

    GitHub push → Vercel 배포

    실제 URL을 받아 학부모·학생에게 공유합니다.

  7. 7

    remotion-video로 소개 영상 제작 (심화)

    완성된 앱을 60초 홍보 영상으로 만들어 공유 채널에 업로드합니다.
